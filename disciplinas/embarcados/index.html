Embarcados

Sistemas Embarcados Link para o cabeçalho

Cada vez mais convivemos com sistemas embarcados e nem percebemos. Seu uso se tornou ubíquo. Além disso, surgiram ferramentas de baixo custo e fácil uso para o seu desenvolvimento. Seja em hardware com o Arduino e módulos ESP, quanto em software, que atualmente são abundantes.

Softwares: Link para o cabeçalho

  • Library.io - Este site é da mesma empresa que desenvolve o software Autocad, a Autodesk. Neste site você possui um Arduino virtual com alguns componentes mais comuns e pode programar como se utilizasse a suite oficial (citada abaixo). É uma excelente ferramenta para ensino e desenvolvimento de protótipos;
  • Arduino - Suite para programação de placas Arduino e também de outras placas cujo suporte é adicionado ao programa através de extensões, como tanto o ESP8266 e ao ESP32;
  • Kicad - Software para criação do esquemático e layout de placa de circuito impresso. Comparável com a ferramenta paga Altium;
  • Fritzing - Software de fácil uso para criação de esquemático e layout de placa, permite a criação de imagens com protoboard para ilustrar guias e relatórios.
  • Freecad - Software para modelagem 3D de peças para posterior impressão. Infelizmente, ao contrário do Kicad fica devendo muito a alternativas pagas, como o Autodesk Inventor (que possúi uma versão gratuita para estudantes).

Sites: Link para o cabeçalho

Diversos sites podem ser de grande ajuda para se atualizar sobre o tema de sistemas embarcados. Estes sites falam sobre lançamentos de placas de circuitos integrados, além de tutoriais de uso dos mais diversos componentes, mas lembre-se a melhor fonte de informações para os componentes são seus respectivos datasheets.

  • Embarcados - Site nacional com notícias de tutoriais relacionados a sistemas embarcados;
  • Arduino e Cia - Site nacional com tutoriais de uso de Arduino e os componentes, sensores e atuadores, mais utilizados;
  • Hackaday - Site internacional com notícias, curiosidades e lançamentos relacionados a sistemas embarcados;
  • CNXSOFT - Site internacional com notícias de lançamentos de hardwares;
  • FCC ID Database - Site com informações de produtos que são homologados pela FCC e Anatel. É possível pesquisar pelo código que está no produto e verificar especificações e fotos internas/externas.

Compra e fabricação de hardware: Link para o cabeçalho

  • OSHPARK - Empresa americana que fabrica placas de circuito impresso (PCBs) e envia para o Brasil. Na minha experiência possuiem um ótimo custo benefício mesmo considerando custo de frete e taxas de importação.
